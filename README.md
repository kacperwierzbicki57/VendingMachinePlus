# ATM & Vending Machine  
**Hybrid Machine with Multi-Currency Support**

This project is an innovative hybrid system that combines the core functionalities of a traditional **ATM** and a **vending machine** into a single intelligent application. Users can select their desired operation at startup and are guided through intuitive, purpose-built interfaces for each mode.
